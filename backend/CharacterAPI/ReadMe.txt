Here are the available routes so far:

GET:
Get all characters:
http://localhost:9018/api/character/
Get character by ID:
http://localhost:9018/api/character/Id

POST:
Create new character(include everything but an ID, which will be auto assigned):
http://localhost:9018/api/character/

PUT:
Update character:
(Pass in battle character object)
http://localhost:9018/api/character/

DELETE:
Delete Character By Id:
http://localhost:9018/api/character/ID
